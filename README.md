# Caravan

Social event platform and free-ticket reservation solution for groups and communities.

## Getting Started

### Prerequisite

Download and install [docker compose](https://docs.docker.com/compose/install/)

### Run containers

From the repository's root directory, run command in terminal:

```bash
docker compose up -d
```

## Local Setup

### Prerequisite

1. Download and install **.NET** SDK 8
2. Clone repository

### Build and run

From the repository's root directory, run command in terminal:

```bash
 dotnet run --project .\Caravan.API\
```

### Access Open API docs

Browse [http://localhost:5259/swagger/index.html](http://localhost:5259/swagger/index.html)
